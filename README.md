## 專案說明
以養雞場為主題的 **單頁式** 形象網站 (純切版)。

做此專案的主要目的：

- 練習 SASS/SCSS 、RWD 以及 jQuery 的使用
- 練習 如何應用套件 (輪播圖、燈箱效果)




## Demo
- https://jhihying.github.io/eggfarm/


## 使用技術 / 套件
- 以手刻的方式來開發 UI 介面
- RWD 斷點：576、768、992、1200 (px)

【使用】HTML5、CSS3、SASS (SCSS)、jQuery、RWD

【套件應用】Swiper (輪播圖)、AOS (視覺滾差動畫)、Lightbox 2 (燈箱)
